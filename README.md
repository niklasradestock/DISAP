# DISAP Demo Prototyp

Objektorientierte Demo für die Endanwender Sicht der DISAP Anwendung.

## Dateien

- `index.html`
- `css/styles.css`
- `js/app.js`

## Funktionen

- Startseite mit Punktekonto und Shopliste
- Aktionsradar mit interaktiver Map und Event Sidebar
- Ankündigungen der Stadt Leipzig mit Informationen, Umfragen und Events
- QR Code Bereich
- Profil Bereich
- Gemeinsame Fußnavigation über Hash Routen

## Klassen in `js/app.js`

- `DemoApp`
- `HomeView`
- `RadarView`
- `AnnouncementsView`
- `QRView`
- `ProfileView`

## GitHub Pages

Die Dateien können direkt in das Repository `DISAP` kopiert werden. Danach unter Settings > Pages den Branch `main` und Root aktivieren.
